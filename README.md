## OPENCV BUILD FARM

#### DEVELOPER TARGET 

* any deb based Linux 
* any rpm based Linux
* macos 10.11 and newer with clang installed


#### BUILD && INSTALL (FOR DEVELOPER ENVS) :

		git clone --recursive git@gitlab.i-free.com:k.shabordin1/opencv-dist.git
		cd opencv-dist
		./prereq.sh
		./build.sh install 


#### BUILD && CREATE DEPLOY PACK 

		git clone --recursive git@gitlab.i-free.com:k.shabordin1/opencv-dist.git
		cd opencv-dist
		./prereq.sh
		./build.sh pack
